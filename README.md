# kkotha19 Stroop Task Experiment

Click the Link to Start the Stroop Task Experiment:
[Stroop Task](https://kkotha19.github.io/stroop_task.html)

This project investigates how cognitive interference affects reaction time and response accuracy using a modified Stroop task. Participants complete three conditions: congruent word-color recognition, incongruent word recognition, and incongruent color recognition. The application randomizes the condition order and anonymously records reaction time and accuracy.

This project was created with the assistance of Claude AI.
